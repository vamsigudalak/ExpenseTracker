# Expense Tracker (React Native CLI)

A simple budget app to track:

- Money In 💰
- Money Out 💸
- Balance 💎

## Features

- Add income & expenses
- Auto calculation
- Clean UI with animations

## Tech Stack

- React Native CLI
- Reanimated
- Vector Icons
